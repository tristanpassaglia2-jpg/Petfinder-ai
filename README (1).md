# PetFinder AI 🐾

**Te ayudamos a encontrar tu mascota en 24-72 horas.**

Plataforma con IA que rastrea en 10+ redes sociales, WhatsApp Bot vecinal, alertas geolocalizadas y difusión inteligente.

## Servicios

- 🔍 **Face ID Tracker** — Reconocimiento visual con IA
- 💬 **WhatsApp Bot** — Avistamientos vecinales sin app
- 📍 **Alertas geolocalizadas** — Radio de búsqueda dinámico
- 📢 **Difusión en redes** — Publicación automática optimizada
- 🔄 **Cruce automático** — Matchea con publicaciones existentes
- 📋 **Avisos IA** — Flyers y textos optimizados

## Stack

- React 18 + Vite
- Claude Sonnet API (Vision + Web Search)
- Stripe + MercadoPago
- Vercel hosting

## Deploy

```bash
npm install
npm run dev     # desarrollo local
npm run build   # build para producción
```

## Variables de Entorno

Copiá `.env.example` como `.env` y completá tus keys.
